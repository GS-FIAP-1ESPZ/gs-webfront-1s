# Space Connect - Global Solution FIAP

Landing page desenvolvida para as disciplinas de **Front-End Design** e
**Web Development** (1º Semestre).

O tema é baseado na **Global Solution da FIAP**: o uso da **tecnologia espacial**
(satélites, dados e inteligência artificial) para resolver desafios da Terra,
como sustentabilidade, agricultura, conectividade e prevenção de desastres.

## Como executar

Abrir o arquivo `index.html` no navegador.

## Seções (Front-End Design)

1. **Problema** – o desafio que a solução resolve
2. **Tecnologia** – tecnologias utilizadas
3. **Objetivos** – principais objetivos do projeto
4. **Público-alvo** – quem é impactado
5. **Benefícios** – ganhos oferecidos
6. **Aplicação no dia a dia** – uso prático da solução

Recursos: menu com indicação da seção ativa, Google Fonts, variáveis CSS,
reset CSS, layout em Flexbox, imagens com `alt`, hovers e rodapé padronizado.

## Funcionalidades (Web Development)

- Slideshow com 3 imagens do tema
- Quiz com 10 perguntas e resultado - validação que impede envio incompleto
- troca de cor de fundo
- JavaScript e responsivo
